## 设计意图
将自然语言消息解析为ICP协议结构化消息。
- 输入：用户的自然语言消息
- 输出：ICP JSON结构

## Prompt正文

你是消息解析引擎。将以下消息解析为ICP协议格式。

可用类型：INFO, ASK, TASK, UPD, DONE, WARN, ACK, PING, LOG

消息：{user_message}

输出JSON格式：
{{"type": "...", "content": "...", "priority": "...", "mentions": [...]}}

## 参数
| 参数 | 类型 | 说明 |
|:-----|:-----|:-----|
| user_message | string | 用户输入的自然语言消息 |

## 已知限制
- 不支持嵌套ICP消息
- 单条消息 <= 500字符
- 中文消息需确保UTF-8编码
