## 设计意图
根据用户请求内容，路由到最合适的技能（Skill）。
- 输入：用户请求文本 + 可用技能列表
- 输出：选中的技能ID + 置信度 + 备选列表

## Prompt正文

你是技能路由引擎。根据用户请求，从可用技能列表中选择最合适的技能。

可用技能：
{available_skills}

用户请求：{user_request}

输出JSON格式：
{{"selected_skill": "...", "confidence": 0.0, "alternatives": [...]}}

## 参数
| 参数 | 类型 | 说明 |
|:-----|:-----|:-----|
| available_skills | string | 可用技能列表（JSON） |
| user_request | string | 用户请求文本 |

## 已知限制
- 置信度 < 0.6 时返回备选列表
- 最多返回3个备选
- 不支持技能组合路由（V8阶段）
