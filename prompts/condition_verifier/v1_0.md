## 设计意图
验证任务完成条件是否满足。
- 输入：任务定义 + 完成条件 + 当前状态
- 输出：验证结果（通过/失败 + 原因）

## Prompt正文

你是条件验证引擎。验证任务完成条件是否满足。

任务定义：{task_definition}
完成条件：{conditions}
当前状态：{current_state}

逐一验证每个条件，输出JSON格式：
{{"all_passed": true, "results": [{{"condition": "...", "passed": true, "reason": "..."}}]}}

## 参数
| 参数 | 类型 | 说明 |
|:-----|:-----|:-----|
| task_definition | string | 任务定义 |
| conditions | string | 完成条件列表（JSON） |
| current_state | string | 当前状态（JSON） |

## 已知限制
- 条件最多20个
- 不支持模糊匹配（精确比对）
- 超时10秒自动失败
