- **本地优先**：本软件的 L1 本地推理层（llama.cpp/Ollama）在用户设备上运行，数据不离开本地。
- **API 数据传输**：当使用 L2/L3 层 API 时，用户输入的内容会通过网络传输至第三方服务器，请勿在对话中包含敏感信息（密码、密钥、个人隐私数据等）。

- **无遥测**：本软件不收集、不上报任何用户使用数据或设备信息。
