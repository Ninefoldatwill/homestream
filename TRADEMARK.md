## When You Can Use the Trademark

You may use the "HomeStream" or "铸钥匠" name without written permission in these cases:

1. **Attribution** — Stating that your project "works with HomeStream" or "built on HomeStream"
2. **Education** — Tutorials, blog posts, conference talks that don't imply endorsement
3. **Open source forks** — As long as it's clear the fork is not the official distribution
4. **Community plugins/extensions** — Descriptive use like "HomeStream Discord Plugin"

## When Permission is Required

You **must** obtain written permission before:

1. Using "HomeStream" or "铸钥匠" in your product, service, or company name
2. Using the HomeStream / 铸钥匠 logo as your own project's logo or favicon
3. Registering domain names containing "HomeStream" or "铸钥匠"
4. Creating merchandise with the HomeStream brand
5. Using the trademark in a way that implies official endorsement

## Guidelines

- **Always attribute**: "HomeStream and 铸钥匠 are trademarks of 九重工作室 (JiuChong Studio)"
- **Don't modify**: Do not alter the logo, change colors, or add effects
- **Be clear**: If you're not the official project, say so explicitly
- **No confusion**: Your use should not confuse users about the origin of the software
